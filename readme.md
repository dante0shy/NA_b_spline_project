##BSpline visualizer

A simple visualizer for B-Spline algorithmn

Requirement:

    Python 3.7

Install:

    git clone https://github.com/dante0shy/NA_b_spline_project.git
    cd NA_b_spline_project
    pip install -r requirement.txt
